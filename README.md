# Driving Social – Community per appassionati di guida

App web e mobile (PWA) per condividere percorsi, foto e mappa, accessibile da browser e da app Android/iOS.

## Stack
- Backend: Python + FastAPI + PostgreSQL (Docker)
- Frontend: Next.js (React)
- Hosting: Debian + Docker Compose

## Come avviare in locale

```bash
docker compose up --build
